#  Универсальный Парсер Маркетплейсов (PARSER67)

Веб-приложение на базе Flask для автоматизированного поиска и агрегации товаров с популярных российских маркетплейсов и досок объявлений.

##  Возможности

- **Поиск по 4 крупным площадкам**: Авито, Яндекс Маркет, Ozon, Wildberries.
- **Гибкие фильтры**: Поиск по ключевым словам, ограничение цены (от и до), выбор состояния товара (б/у или новый — для Авито), выбор города.
- **Обход защиты от ботов**: Использование stealth-технологий (на базе `undetected-chromedriver`) для имитации действий реального пользователя и обхода блокировок.
- **Выгрузка данных**: Возможность скачать результаты поиска в удобном формате Excel (`.xlsx`) одним кликом прямо из веб-интерфейса.
- **Современный интерфейс**: Удобный и минималистичный темный UI-дизайн.
